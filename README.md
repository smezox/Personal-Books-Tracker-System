# Personal-Books-Tracker
The Personal Book Tracker System is a mini project developed using Java and Object-Oriented Programming (OOP) concepts. The main purpose of this project is to help users keep track of the books in their personal library and monitor their reading progress. The system allows the user to add multiple books, store details such as title, author, category, rating, start date, finish date, and review their books, and organize the books based on their reading status such as Not Started, Reading and Finished. This project is designed using core OOP principles like encapsulation, classes, objects, and ArrayList collections to manage data efficiently. Separate classes such as Book, BookTracker, and Main are used to maintain modularity and proper structure of the program. The application provides an interactive menu-driven interface where users can add books, update reading status, view their library, and count completed books. The Book Tracker System demonstrates how object-oriented programming can be used to build a simple but practical application that manages data in an organized and user-friendly way. This project helps in understanding the real-world use of Java OOP concepts and improves programming skills in designing structured applications. Many existing book tracking solutions are either large library management systems or online applications that require internet access, user accounts, or complex interfaces. These systems are often designed for managing large public libraries rather than personal reading collections. In contrast, this Book Tracker System is a lightweight, console-based personal library manager designed specifically for individual users. It runs locally without requiring internet connectivity or external databases. Unlike many simple book lists, this system also tracks reading progress, reading dates, and personal notes, providing a more complete view of a user's reading activity. Additionally, the system is developed using Java Object-Oriented Programming concepts, making it a practical educational project that demonstrates how real-world applications can be structured using classes and modular design.
